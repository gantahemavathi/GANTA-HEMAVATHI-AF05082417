# GANTA-HEMAVATHI-AF05082417
HR ANALYTICS DASHBOARD
Dataset Description

The dataset used in this project contains employee information related to workforce demographics, job roles, salary, work experience, and job satisfaction. It includes multiple attributes such as employee age, gender, department, job role, monthly income, total working years, years at the company, work-life balance, and attrition status. This data helps in analyzing employee behavior and identifying patterns related to employee retention and attrition within the organization.

Dashboard Development

The dashboard was developed using Power BI Desktop by importing the dataset and transforming it into meaningful visual insights. Key performance indicators such as total employees, attrition count, attrition rate, and average income were created to provide a quick overview of workforce statistics. Different visualizations such as bar charts, pie charts, line charts, and tables were used to analyze employee distribution across departments, job roles, age groups, and salary levels. Filters and interactive elements were added to allow users to explore the data and gain deeper insights into employee trends.

Conclusion

Power BI is an effective tool for transforming raw data into interactive dashboards and meaningful visual reports. By using Power BI in this project, complex employee data was converted into clear visual insights that help understand workforce patterns and attrition trends. This demonstrates how data visualization can support better decision-making in organizations.
